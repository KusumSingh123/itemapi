# Item Management REST API

## Description
This is a simple Spring Boot REST API to manage items (like an e-commerce product system).

## Features
- Add item
- Get item by ID
- In-memory storage using ArrayList
- Input validation

## Technologies
- Java
- Spring Boot
- Maven

## API Endpoints

### Add Item
POST /api/items

### Get Item by ID
GET /api/items/{id}

## Run Application
mvn spring-boot:run
